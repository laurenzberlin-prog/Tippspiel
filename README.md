# Tippspiel

Projekt für die Design Challenge E: Tippspiel.

Die vollständige Dokumentation ist [hier](https://laurenzberlin-prog.github.io/Tippspiel/) zu finden. 


## Quick Start

### Vorraussetzungen:

-   Python 3.7 oder neuer
-   Pip

### Klone das repository:

-   https://github.com/laurenzberlin-prog/Tippspiel

### Dependencies installieren

-   pip install -r requierments.txt

### Ausführen der Anwendung

-   python app.py

### Zugang zu Anwendung

-   Im Browser öffnen: http://127.0.0.1:5000
